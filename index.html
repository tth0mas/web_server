Introduction

The Greatest Common Divisor

The greatest common divisor is the largest integer which divides two numbers.

[quote:def1] Definition 1: An integer b is said to be divisible by an integer a ≠ 0, often expressed as a|b, if there exists some integer c such that b = ac. We write a ∤ b to indicate that b is not divisible by a.

From definition [quote:def1] the following hold for integers a, b and c.
a|0, 1|a, a|a

A divides zero can be gathered quite easily from looking at it.
$$a|0 = \frac{a}{0} = 0 \ 0 = a \cdot 0,$$
which holds for all a.

Definition 2: Let a and b be given integers, with at least one of them different from zero. The greatest common divisor of a and b, denoted gcd(a,b) is the positive integer d satisfying the following:

  • d|a and d|b

  • if c|a and c|b then c ≤ d

Given the integers a and b, we say that the two integers a and b are relatively prime if gcd(a,b) = 1, meaning the only common divisor both integers have is 1.

The Division Algorithm

[teor:da] Theorem 1 Given integers a and b, with b > 0, there exists unique integers q and r satisfying


a = qb + r   0 ≤ r < b

Stating that there exists integers q and r, called quotient and remainder, respectively in the division of a by b.

Euclids Algorithm

Let a and b be two integers. We know that


gcd(|a|,|b|) = gcd(a, b),
and as such, can assume a and b to be positive integers. If the division algorithm [teor:da] is applied to a and b


a = q1b + r1   0 ≤ r1 < b
If r1 is zero, the algorithm stops because b|a and gcd(a,b) would be b. If r1 ≠ 0, we follow the next step of the algorithm by moving b to the place of a in the previous step
b = q2r1 + r2   0 ≤ r2 < r1
If the remainder r2 is zero, we stop and the result of the algorithm is r1, i.e., gcd(a,b) = r1. If not, the next step is


r1 = q3r2 + r3   0 ≤ r3 < r2
We can continue in this manner until we reach a zero remainder. When we reach zero we can say that the last non-zero remainder is equal to the gcd(a,b). If the remainder rn is zero, this would correspond to rn − 1 being the gcd(a,b).

Bézouts Identity

Bézouts states that the gcd of two number can be expressed as a linear combination of both numbers

[teor:bez] Theorem 2 If a and b are positive integers, then there exists integers s and t such that gcd(a,b) = sa + tb. The integers s and t are called Bézout coeffiients.

Large Primes

Theorem 3 Every positive integer n > 1 is either a prime or a product of primes; this representation is unique, apart from the order in which the factor occur.

called the fundamental theorem of arithmetic.

A corollary from the fundamental theorem of arithmetic is that

Any positive integer n > 0 can be written uniquely in a cannonical form
n = p1k1p2k2p3k3...prkr,
where for i = 1,2,3... r, each ki is a positive integer and each pi is a prime, with p1 < p2 < p3... < pr.

If we allow for zero exponents, this gives us another way to calculate the gcd. Given
a = p1k1p2k2p3k3...pnkm
and
b = p1j1p2j2p3j3...pnjn,
then
gcd(a, b) = p1r1p2r2p3r3...pnrn,
where ri = min(ki, ji).
